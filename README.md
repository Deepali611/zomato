# Zomato AI Restaurant Recommendations

AI-powered restaurant discovery and smart recommendations concierge.

## Project Structure
* `/frontend`: Next.js Web UI
* `/src`: Python Backend API
* `/data`: Restaurant Data Parquet database

---

## 🎨 Frontend Setup (Next.js)
The frontend is located in the `frontend` folder.

```bash
cd frontend
npm install
npm run dev
```

---

## ⚙️ Backend Setup (Python)
The backend is located at the root of the project.

```bash
.python-embed/python.exe -m pip install -r requirements-backend.txt
.python-embed/python.exe src/presentation/api.py
```
